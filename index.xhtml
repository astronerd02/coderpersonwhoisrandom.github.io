How to simplify Radicals


So you want to simplify radicals? If not, go somewhere else, this is not for you. But for everybody else, we will start with explaining what a radical is.

So this is a radical: 4, which is just equal to 2 since 4 is a square number. It is just a number under the square root sign. Most radicals are like this, 8, which is a nasty irrational number. Instead of trying to find the nasty decimal, you can simplify the radical. The content below is a step by step guide on how to simplify a radical, in this case, 8

Simplifying 8

  1. Find the factors of 8, (2, 4) and (1, 8)
  2. Look for a factor pair with a square number, in this case (2, 4) because 4 is a square number
  3. Don't forget that you are working with radicals
  4. Multiply 4 and 2
  5. Reduce 4 to 2
  6. You are multiplying 2 and 2, and the product is this: 22
  7. Double check your math

Now for the breakdown of what happened. So first off, you CAN multiply 2 radicals, such as 2 · 4 = 8. So the first step is self-explanitory, you are just finding the factors of the number under the square root sign. The next step exists because you can only simplify radicals with a square number in their factors, and if there is none, it is already simplified. If you were doing it correctly and used a square number in your factors, reduce the square number by finding the square root either by memory or calculator. In the example above it was 4 · 2, you would reduce 4 to 2, multiply 2 · 2 to get 22. And that is how you simplify square roots!

Simplifying Radicals with variables

So now you know (hopefully) how to simplify radicals with only whole numbers (negative numbers have no square roots), but you may see a variable from time to time. So to start with the basics, lets view x2. So you see it has the exponent 2, which means it is x squared. So here is some knowledge, squaring and square rooting are inverse opperations, meaning if you do both to a number or variable, you would end up where you started. You are doing that to x, so the answer is x. But what about x3? The first step to solving this is to break down x3. So the exponent 3 means you are multiplying 3 x's together, so the broken down verison is x · x · x. We already established that x2 = x, so for every two x's under the radical sign, we can bring one x in front of it. Because there are 3 x's under the radical sign, we can replace two of the x's under the sign for 1 x outside of the sign, and leave the leftover x under there to get us our final answer of xx. Now that you know what is going on, now for a shortcut. divide the exponent by 2, if it is an even number, there will be nothing left beneath the radical sign, but id it is odd, there will always be one x under the radical sign.

Well, those were the basics! We're not going to get into the complicated stuff so enjoy your new knowledge!
